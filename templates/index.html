API使用说明

本工具提供了简单易用的API接口，可以集成到您的应用程序中，实现自动化的curl命令解析和转换。

接口地址

使用以下HTTP请求方法和路径访问接口：

POST /api/convert

请求格式

请求体应为JSON格式，包含一个curl字段，值为要解析的curl命令字符串：

{
    "curl": "your curl command here"
}

响应格式

响应内容为结构化的JSON对象，包含以下字段：

  • method: HTTP请求方法
  • url: 请求URL
  • headers: 请求头信息
  • data: 请求数据
{
    "method": "POST",
    "url": "https://api.example.com",
    "headers": {
        "Content-Type": "application/json"
    },
    "data": {
        "key": "value"
    }
}

示例代码

以下是不同编程语言中调用本工具API的示例代码：

fetch('/api/convert', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        curl: 'curl -X POST https://api.example.com -H "Content-Type: application/json" -d \'{"key":"value"}\''
    })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
import requests
import json

url = "http://localhost:5000/api/convert"
payload = {
    "curl": 'curl -X POST https://api.example.com -H "Content-Type: application/json" -d \'{"key":"value"}\''
}
headers = {
    "Content-Type": "application/json"
}

response = requests.post(url, headers=headers, data=json.dumps(payload))
print(response.json())
curl -X POST http://localhost:5000/api/convert \
  -H "Content-Type: application/json" \
  -d '{"curl": "curl -X POST https://api.example.com -H \"Content-Type: application/json\" -d \"{\\\"key\\\":\\\"value\\\"}\""}'

关于Curl转JSON工具

Curl转JSON工具是一个专为开发者设计的在线实用工具，旨在简化API调试和HTTP请求分析过程。通过将复杂的curl命令转换为结构化的JSON格式，开发者可以更轻松地理解和处理HTTP请求。

主要功能

  • 快速转换 - 将curl命令即时转换为JSON格式
  • 全面解析 - 支持多种curl参数，如请求方法、头信息、数据等
  • API接口 - 提供简单易用的API，便于集成到其他系统
  • 用户友好 - 直观的界面设计和交互体验

适用场景

  • API开发和测试
  • 网络请求调试
  • 自动化脚本编写
  • 学习HTTP协议
