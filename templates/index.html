Examples

Here are some common examples of using our conversion tools:

GET Request

curl -X GET https://api.example.com/users

Converts to:

{
    "method": "GET",
    "url": "https://api.example.com/users",
    "headers": {}
}

POST with JSON

curl -X POST https://api.example.com/users \
  -H 'Content-Type: application/json' \
  -d '{"name":"John","email":"john@example.com"}'

Converts to:

{
    "method": "POST",
    "url": "https://api.example.com/users",
    "headers": {
        "Content-Type": "application/json"
    },
    "data": {
        "name": "John",
        "email": "john@example.com"
    }
}

Python (requests)

import requests

url = "https://api.example.com/users"
headers = {"Content-Type": "application/json"}
data = {"name": "John", "email": "john@example.com"}

response = requests.post(url, headers=headers, json=data)

Converts to:

curl -X POST 'https://api.example.com/users' -H 'Content-Type: application/json' -d '{"name":"John","email":"john@example.com"}'

JavaScript (fetch)

fetch('https://api.example.com/users', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        name: 'John',
        email: 'john@example.com'
    })
})

Converts to:

curl -X POST 'https://api.example.com/users' -H 'Content-Type: application/json' -d '{"name":"John","email":"john@example.com"}'

API Documentation

This tool provides simple and easy-to-use API interfaces that can be integrated into your application for automated conversions.

Endpoints

Use the following HTTP request methods and paths to access the APIs:

Curl to JSON

POST /api/convert

Code to Curl

POST /api/reverse

Request Format

The request body should be in JSON format, containing a curl field with the curl command string to be parsed:

{
    "curl": "your curl command here"
}

The request body should be in JSON format, containing code and language fields:

{
    "code": "your code here",
    "language": "python" // or "javascript", "php"
}

Response Format

The response content is a structured JSON object containing the following fields:

  • method: HTTP request method
  • url: Request URL
  • headers: Request headers
  • data: Request data
{
    "method": "POST",
    "url": "https://curl-to.com",
    "headers": {
        "Content-Type": "application/json"
    },
    "data": {
        "key": "value"
    }
}

The response content is a JSON object containing the following fields:

  • command: The generated curl command
  • success: Boolean indicating if conversion was successful
  • message: Error message if conversion failed
{
    "command": "curl -X POST 'https://api.example.com' -H 'Content-Type: application/json' -d '{\"key\":\"value\"}'",,
    "success": true,
    "message": ""
}

.then(response => response.json())

import json

url = "http://curl-to.com/api/convert"
payload = {
    "curl": 'curl -X POST https://curl-to.com -H "Content-Type: application/json" -d \'{"key":"value"}\''

headers = {
    "Content-Type": "application/json"
}

response = requests.post(url, headers=headers, data=json.dumps(payload))
print(response.json())

About Curl to JSON Tool

Curl to JSON Tool is an online utility designed specifically for developers, aimed at simplifying API debugging and HTTP request analysis. By converting complex curl commands into structured JSON format, developers can more easily understand and process HTTP requests.

Key Features

  • Fast Conversion - Convert curl commands to JSON format instantly
  • Comprehensive Parsing - Support for various curl parameters such as request methods, headers, data, etc.
  • API Interface - Provides a simple and easy-to-use API for integration with other systems
  • User Friendly - Intuitive interface design and interaction experience

Use Cases

  • Network Request Debugging
  • Automation Script Writing
