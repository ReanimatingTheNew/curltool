使用示例

了解如何使用Curl转换工具进行各种HTTP请求转换

基本GET请求

Curl命令

curl -X GET https://api.example.com/users

JSON结果

{
  "method": "GET",
  "url": "https://api.example.com/users",
  "headers": {}
}

带JSON数据的POST请求

Curl命令

curl -X POST https://api.example.com/users \
  -H "Content-Type: application/json" \
  -d '{"name":"John","email":"john@example.com"}'

Python代码

import requests

headers = {
    'Content-Type': 'application/json',
}

data = {
    'name': 'John',
    'email': 'john@example.com'
}

response = requests.post('https://api.example.com/users', headers=headers, json=data)
print(response.json())

带认证的请求

Curl命令

curl -X GET https://api.example.com/users \
  -H "Authorization: Bearer token123"

JavaScript代码

fetch('https://api.example.com/users', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer token123'
  }
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));

API文档

了解如何集成Curl转换工具的API到您的应用程序中

API端点

Curl转JSON

URL: /api/convert

方法: POST

描述: 将curl命令转换为JSON格式

Curl转代码

URL: /api/curl-to-code

方法: POST

描述: 将curl命令转换为指定编程语言的代码

代码转Curl

URL: /api/reverse

方法: POST

描述: 将代码转换为curl命令

获取支持的语言

URL: /api/languages

方法: GET

描述: 获取支持的编程语言列表

请求格式

Curl转JSON

{
  "curl": "curl -X GET https://api.example.com/users"
}

Curl转代码

{
  "curl": "curl -X GET https://api.example.com/users",
  "language": "python"
}

代码转Curl

{
  "code": "import requests\n\nresponse = requests.get('https://api.example.com/users')\nprint(response.json())",
  "language": "python"
}

响应格式

Curl转JSON

{
  "method": "GET",
  "url": "https://api.example.com/users",
  "headers": {}
}

Curl转代码

{
  "code": "import requests\n\nresponse = requests.get('https://api.example.com/users')\nprint(response.json())",
  "language": "python"
}

代码转Curl

{
  "success": true,
  "command": "curl -X GET https://api.example.com/users"
}

关于Curl转换工具

了解我们的工具及其功能

项目介绍

Curl转换工具是一个强大的HTTP请求转换器，旨在简化开发人员的API调试和开发工作流程。它可以将curl命令转换为JSON格式或各种编程语言的代码，也可以将代码转换回curl命令。

主要功能

  • 将curl命令转换为结构化JSON格式
  • 将curl命令转换为多种编程语言的代码
  • 将代码转换为等效的curl命令
  • 提供RESTful API接口，方便集成到其他应用
  • 支持多种HTTP方法、请求头、认证和数据格式
